Talkbox

An assistive technology framework based on the MPR121 capacitive touch sensor and RaspberryPi.

  • Download ZIP
  • Download TAR
  • View On GitHub

This project is maintained by hrairhlessil

About

TalkBox is an assistive technology project brought to you by the GaMaY lab at the Lassonde School of Engineering in Toronto. It is about using methodologies and tools available in the hacker and maker communities to create assistive technologies.

Contributing

TalkBox is an open source project and we welcome all contributions and improvements by the community! Feel free to fork and modify.

Install

Setting up your very own TalkBox is easy:

  • Download the TalkBox image. (will update)
  • dd image onto SD card.
  • Plug MPR121 board into GPIO according to schematic
  • Plug into electricity and use
